# IBA Preview

Site mínimo de "em breve / coming soon" para a **Innovation Business Alliance (IBA Network)** — plataforma global conectando FinTechs brasileiras aos mercados financeiros da Irlanda e do Reino Unido.

## Requisitos

- Node.js >= 20.9
- npm

## Scripts

```bash
npm run dev     # http://localhost:3000 (desenvolvimento)
npm run build   # build de produção (estático)
npm run start   # serve o build em http://localhost:3000
```

## Intenção

Landing holding page, 100% texto, sem imagens. Apresenta:

- **Hero**: "A global platform connecting Brazil FinTech to Ireland and the UK financial markets."
- **Vision + Mission**: positioning da IBA
- **What we do**: conectar FinTechs Brasil → Irlanda/UK
- **In the IBA you'll find**: benefícios
- **Join the movement**: canais (social, eventos/trade missions, plataforma proprietária em stealth)
- **Coming soon**: aviso de site completo em construção
- **Supported by**: Brazil Embassy in Dublin, MWeds, Novos Ventures, ABFintechs
- **Contact**: e-mail + LinkedIn
